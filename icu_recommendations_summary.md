# ICU Sheet Design: Evidence-Based Recommendations Summary

## Research Synthesis

**Patient Focus**: Depth over breadth → **1 patient per page** optimal (Etsy bestsellers confirm)

**Critical Fields Must Include**:
- Hemodynamics (BP, HR, MAP, CVP)
- Ventilator settings (mode, FiO2, PEEP, rates)
- Continuous drips (with titration tracking)
- Neuro checks (GCS, pupils)
- Strict I&Os (hourly balances)

**Pain Points to Address**:
- Overcrowding → Simplify layout
- Redundancy → Integrate with EMR
- ADHD challenges → Visual clarity, color coding

## 5 Core Recommendations

### 1. Layout: One Patient Per Page
- **Why**: Research shows 1 patient allows sufficient detail space
- **How**: Portrait orientation, dedicated sections per system
- **Benefit**: Reduces cognitive load, prevents cross-patient confusion

### 2. Hourly Tracking: Tiered Grid System
- **Structure**: 24-column grid with color-coded rows by system
- **Space optimization**: Condensed fonts, abbreviations with legend
- **Visual aids**: Trend arrows, shaded alternating hours, checkbox grids for neuro

### 3. Drip/Vent Documentation: Integrated Flow Sheets
- **Drips**: Medication | Rate | Titration log | Running total
- **Vent**: Setting | Current | Change log | Alarms
- **Visual cues**: Color by drug class (red=vasopressors, blue=sedatives), flow arrows for changes

### 4. ADHD-Friendly Visual Design
- **Color coding**: Red=critical, Blue=respiratory, Green=neuro, Yellow=I&O, Purple=labs
- **Hierarchy**: Bold borders around critical sections, icon indicators for abnormal values
- **ADHD features**: Checklist mentality, visual reminders, minimalist design, high contrast

### 5. Customizability: Template-Based with Guardrails
- **Base template**: All critical fields (non-negotiable for safety)
- **Optional sections**: Turn on/off based on patient needs (trauma, infection, specialty)
- **Digital flexibility**: Drag-and-drop rearrangement, personal shorthand space
- **Unit-specific**: Each ICU can customize within safety limits

## Implementation Priority

1. **Start with paper prototype** → Single-patient 24-hour sheet
2. **Test with nurse focus group** → Iterate based on ADHD usability feedback
3. **Digital adaptation** → Tablet interface with auto-population from EMR
4. **Integration** → Minimize redundancy, one-touch documentation

## Success Metrics Target
- **20% reduction** in documentation time
- **Fewer missed assessments** and medications
- **Nurse preference** for new design
- **Early deterioration detection** improvement

## Key Insight
The most successful ICU sheets (per Etsy market data) focus on **single-patient depth** with **editable templates**. Combine this with research-identified critical fields and ADHD-friendly design principles for optimal results.
