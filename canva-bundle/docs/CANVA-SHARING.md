# How to Share Editable Canva Templates with Customers

## Overview

For **Gumroad** sales, you deliver editable Canva templates by sharing Canva design links. For **Etsy** PDF sales, you deliver exported PNG/PDF files.

This guide covers both methods.

---

## Method 1: Canva Share Links (For Gumroad — Recommended)

### Step-by-Step Instructions

1. **Create your designs in Canva**
   - Recreate each template from the HTML design files
   - Customize the placeholder content with professional stock content
   - Ensure all text, colors, and branding are finalized

2. **Organize your templates**
   - Create a folder in Canva for each template category:
     - `01 Instagram Posts`
     - `02 Instagram Stories`
     - `03 LinkedIn Banners`
     - `04 YouTube Thumbnails`
     - `05 Presentation Slides`

3. **Share each design**
   For **each template**, do the following:
   - Open the design in Canva
   - Click the **"Share"** button
   - Set permissions to **"Anyone with the link can edit"** (recommended for customers)
   - OR set to **"Anyone with the link can view"** if you only want them to see
   - Copy the share link

4. **Create a template access page**
   - Create a simple Notion page, Gumroad update, or PDF
   - List all 41 template links organized by category
   - Include brief instructions for each link

5. **Deliver to customer**
   - Add the link page to your Gumroad product as a bonus/appendix
   - OR send as a follow-up email after purchase
   - OR include in your Etsy PDF as a separate "Bonus: Editable Canva Links" section

### Sharing Settings Explained

| Setting | Best For | Customer Can |
|---------|---------|-------------|
| "Anyone with link can edit" | Full template access | Duplicate, edit, download |
| "Anyone with link can view" | Preview only | See but not edit |
| "Invite people" | Private access | Requires Canva account |

**Recommendation:** Use **"Anyone with the link can edit"** so customers can fully customize. They need their own Canva account to save changes.

## Method 2: PDF/PNG Exports (For Etsy — Required)

### Step-by-Step Instructions

1. **Export each template as PNG/PDF**
   - In Canva, click **"Download"**
   - Choose **PNG** (for images) or **PDF Print** (for documents)
   - Quality: **Best**
   - For PDF: Select "PDF Print" not "PDF Standard"

2. **Organize files into folders**
   ```
   Coach-Bundle/
   ├── Instagram-Posts/     (12 PNG files)
   ├── Instagram-Stories/  (10 PNG files)
   ├── LinkedIn-Banners/   (6 PNG files)
   ├── YouTube-Thumbnails/ (8 PNG files)
   └── Presentations/      (5 PDF files)
   ```

3. **Create a master PDF guide**
   - Combine all PNG files into a PDF
   - Include instructions, tips, and customization guide
   - Add your branding to the cover page

4. **Zip everything**
   - Compress the entire folder structure
   - Name it: `Coach-Template-Bundle-V1.zip`

5. **Upload to Etsy/Etsy PDF**
   - Upload the ZIP file as your product file
   - Etsy will deliver it automatically after purchase

### Export Quality Settings

| Format | Use Case | Quality Setting |
|--------|----------|----------------|
| PNG | Social media posts | Best (highest) |
| PDF Print | Presentations, guides | Best (highest) |
| JPG | Smaller file sizes | Best (highest) |

## How to Set Up Canva Teams (For Managing Multiple Clients)

If you're using these templates to serve clients:

1. Create a **Canva Teams** account ($12.99/month per seat)
2. Create a **Team folder** for each client
3. Copy template designs into each client folder
4. Invite clients as team members
5. They can edit their folder without affecting other clients

**Note:** Canva Teams is for your internal business use. You cannot resell template access to clients as a service.

## Troubleshooting

**Q: Customer can't edit the template**
A: Ensure sharing is set to "Anyone with link can edit." Customer must have a free Canva account to save changes.

**Q: Canva link shows "template not found"**
A: The template may have been deleted. Keep your master templates in a private folder and share duplicates.

**Q: Fonts look different for my customers**
A: Canva automatically substitutes fonts when the original isn't available. Use only Canva's built-in fonts (Montserrat, Open Sans, Playfair Display) which are always available.

**Q: Can I use Canva Teams to deliver to customers?**
A: Not recommended. Canva Teams is for internal collaboration, not customer delivery. Use share links instead.
