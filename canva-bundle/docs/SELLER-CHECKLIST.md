# Seller Checklist — Everything You Need to Launch

## Before You Start

- [ ] Read all documentation files
- [ ] Understand the two delivery methods (Gumroad vs Etsy)
- [ ] Set up Canva account (free at canva.com)

---

## Step 1: Recreate Templates in Canva

For each of the 41 templates in the HTML design files:

1. Open Canva → Create new design with correct dimensions
2. Recreate the layout from the HTML reference
3. Add professional placeholder text (not "Lorem ipsum")
4. Use exact color hex codes from TEMPLATE-SPECS.md
5. Apply correct fonts from TEMPLATE-SPECS.md
6. Save the design in your Canva account

**Dimensions Reference:**
| Type | Width | Height |
|------|-------|--------|
| Instagram Post | 1080px | 1080px |
| Instagram Story | 1080px | 1920px |
| LinkedIn Banner | 1584px | 396px |
| YouTube Thumbnail | 1280px | 720px |
| Presentation | 1920px | 1080px |

**Pro tip:** Work category by category. Recreate all 12 IG posts first, then all 10 stories, etc.

## Step 3: For Etsy (PDF/PNG Downloads)

- [ ] Export all 41 templates from Canva:
  - Download each as PNG (Best quality)
  - Download presentations as PDF Print
- [ ] Create folder structure:
  ```
  Coach-Template-Bundle/
  ├── Instagram-Posts/     (12 PNGs)
  ├── Instagram-Stories/  (10 PNGs)
  ├── LinkedIn-Banners/   (6 PNGs)
  ├── YouTube-Thumbnails/ (8 PNGs)
  ├── Presentations/      (5 PDFs)
  ├── BONUS-Fonts-Colors/ (PDF spec sheets)
  └── README.pdf          (customization guide)
  ```
- [ ] Create a professional PDF README with:
  - How to use the templates
  - Color palette hex codes
  - Font list
  - Link to Canva (free signup)
  - License terms
- [ ] Zip the entire folder (keep under 500MB)
- [ ] Create Etsy listing:
  - Title: `Professional Coach Social Media Template Bundle | 41 Canva Templates | IG, Stories, LI, YouTube`
  - Upload ZIP as file
  - Paste Etsy description from sales-pages/etsy-description.md
  - Set price: $57 (launch at $27)
  - Publish

## Common Mistakes to Avoid

❌ **Don't** just upload the HTML files — customers need Canva/PNG files  
❌ **Don't** forget to set sharing permissions to "Anyone can edit"  
❌ **Don't** price too low ($19 signals low quality; $67 signals premium)  
❌ **Don't** skip the PDF guide — it adds perceived value  
❌ **Don't** launch without testing your Canva links first  
❌ **Don't** use lorem ipsum — use real, compelling placeholder text
