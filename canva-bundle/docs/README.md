## 🎨 Design Systems

### Color Palettes

**Bundle includes 3 complete color palettes:**

1. **Midnight Gold** — Sophisticated dark theme with gold accents
   - Primary: `#0D0D0D` | Accent: `#D4AF37` | Text: `#FFFFFF`

2. **Sage Serenity** — Calming earthy greens
   - Primary: `#4A6741` | Accent: `#C9A96E` | Text: `#2C2C2C`

3. **Ocean Depth** — Bold blue professional
   - Primary: `#1A365D` | Accent: `#4299E1` | Text: `#FFFFFF`

### Typography

- **Headlines:** Montserrat (Bold/Black)
- **Subheadings:** Montserrat (SemiBold)
- **Body:** Open Sans (Regular/Light)
- **Accents:** Playfair Display (Quotes, highlights)
