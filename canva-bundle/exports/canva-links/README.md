# Canva Editable Links — READ ME FIRST

## IMPORTANT: How to Create Canva Links

Unfortunately, **Canva does not provide a public API** for creating designs programmatically. Each template must be recreated in Canva manually using the PNG/PDF files in this folder as a visual reference.

### Why These Aren't Pre-Made Canva Links?

- Canva's API requires enterprise authentication and is not available to individual developers
- Each Canva account is separate, so links must be created within YOUR Canva account
- This ensures buyers get EDITABLE templates, not just static images

## Instructions: How to Create Editable Canva Templates

### Step 1: Open a Template Image
Open any PNG file from this folder to see the exact design you need to recreate.

### Step 2: Create in Canva
1. Go to [canva.com](https://www.canva.com) and log in (or create free account)
2. Click **"Create a design"**
3. Choose the appropriate dimensions:
   - Instagram Posts: 1080 × 1080 px
   - Instagram Stories: 1080 × 1920 px
   - LinkedIn Banners: 1584 × 396 px
   - YouTube Thumbnails: 1280 × 720 px
   - Presentation Slides: 1920 × 1080 px

### Step 3: Recreate the Design
Use Canva's built-in tools to recreate each design:
- Add text elements matching the template
- Use shapes and lines for decorative elements
- Apply the color palette (see COLORS.md)
- Use the fonts listed in FONTS.md

### Step 4: Share the Editable Link
1. Click **"Share"** button in Canva
2. Change permissions to **"Anyone with link can edit"**
3. Copy the link and save it to the corresponding file in this folder
