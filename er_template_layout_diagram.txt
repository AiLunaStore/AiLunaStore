Key Design Features:
• Two‑column layout maximizes detail per patient while keeping page count to one.
• Sections flow top‑to‑bottom by clinical priority (urgent data first).
• Color‑coding ties to ESI level for instant visual acuity recognition.
• High‑frequency vitals grid supports trauma‑style monitoring (q5‑15min).
• SBAR handoff built‑in promotes structured communication.
• Trauma team roles appear only when trauma activation is checked.
• Minimal long‑term tracking (no hourly I&O) – focus on acute trends.

Print Notes:
• Light background tints (≤10% saturation) conserve ink.
• All borders are 0.5 pt lines.
• Font sizes: headers 10 pt, body 8 pt.
• Margins 0.5″ on all sides.
