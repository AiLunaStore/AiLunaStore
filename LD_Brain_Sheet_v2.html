10. Fetal Heart Rate Categories (NICHD Guidelines)
Category I - Normal
All of the following present:
• Baseline: 110-160 bpm
• Variability: Moderate
• Accelerations: Present or absent
• Decelerations: None or early
→ No intervention needed
Category II - Indeterminate
Any pattern not Category I or III:
• Bradycardia w/ variability
• Tachycardia
• Minimal/marked variability
• Accelerations: Absent
• Variable decels: >50% of contractions
• Prolonged decels: >2 min
• Slow return to baseline
→ Evaluate, monitor, position, O2
Category III - Abnormal
Any of the following:
• Sinusoidal pattern
• Absent variability WITH:
  - Recurrent late decels
  - Recurrent variable decels
  - Prolonged decels >10 min
→ IMMEDIATE EVALUATION
→ Notify provider, prepare for delivery
Interventions for Category II/III:
1. Position change (lateral) → 2. O2 10L mask → 3. IV bolus → 4. Stop Pitocin → 5. Check FHR correlation → 6. Notify provider
