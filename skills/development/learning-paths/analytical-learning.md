## Learning Path: Research

### Phase 1: Information Gathering (Complete)
**Status:** ✅ Complete

**Competencies:**
- Web search effectively
- Evaluate sources
- Extract key information
- Organize findings

**Validation:**
- Can find relevant information quickly
- Can assess source credibility
- Can summarize effectively

### Phase 2: Synthesis (Complete)
**Status:** ✅ Complete

**Competencies:**
- Connect information sources
- Identify gaps
- Build coherent narratives
- Cite appropriately

**Validation:**
- Can integrate multiple sources
- Can identify what's missing
- Can present findings clearly

### Phase 3: Critical Analysis (In Progress)
**Status:** 🔄 Active

**Competencies:**
- Evaluate arguments
- Identify biases
- Assess methodology
- Question assumptions

**Current Focus:**
- Source credibility assessment
- Bias detection

**Validation:**
- Can identify weak arguments
- Can spot logical fallacies
- Can assess research quality

### Phase 4: Expert Research (Planned)
**Status:** 📋 Planned

**Competencies:**
- Design research strategies
- Use specialized databases
- Conduct original analysis
- Publish findings

### Phase 5: Research Leadership (Target)
**Status:** 🎯 Target

**Competencies:**
- Create research methodologies
- Train researchers
- Set research standards
- Pioneer new approaches
