# Skill Self-Assessment

## Purpose
Evaluate current skill levels across all domains to identify strengths and development opportunities.

## Assessment Instructions

1. **Rate each skill** on the 1-5 scale below
2. **Be honest** - this is for your development
3. **Provide evidence** for ratings
4. **Identify gaps** between current and target levels
5. **Prioritize** development areas

## Rating Scale

| Rating | Level | Description |
|--------|-------|-------------|
| 1 | Novice | Basic understanding, needs guidance |
| 2 | Competent | Can use independently, occasional reference |
| 3 | Proficient | Efficient, knows best practices |
| 4 | Expert | Deep mastery, can teach others |
| 5 | Master | Innovation, creates new approaches |
