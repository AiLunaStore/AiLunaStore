# GitHub Management

GitHub Management covers the practices and tools for organizing repositories, managing issues, reviewing pull requests, and automating workflows using the `gh` CLI. Effective GitHub management includes structuring repos for discoverability, using labels and milestones consistently, and automating CI/CD pipelines. Skills like the `gh-issues` and `github` skills enable AI-driven issue triage and PR automation. It connects to [[Product-Development]] as the version control backbone for shipping features, and [[OpenClaw-Setup]] for maintaining workspace dotfiles and configuration in Git.
