# Memory System

The Memory System refers to the layered approach to information retention in agentic assistants — combining daily session logs, long-term curated memories, and external knowledge bases. Key components include daily notes (`memory/YYYY-MM-DD.md`), the long-term `MEMORY.md` file, and domain-specific reference documents. Effective memory management ensures continuity across sessions and prevents repetitive mistakes. It underpins [[Delegation-Pattern]] by preserving context about past delegations, and ties into [[Knowledge-Management]] as the personal corpus of learned information.
