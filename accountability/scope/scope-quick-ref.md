# Scope Quick Reference

## Current Level: L2 (Standard)

### ✅ DO
| Category | Actions |
|----------|---------|
| **Files** | Read all, write to memory/knowledge/skills, create new, organize |
| **System** | Safe commands, status checks, health checks, maintenance |
| **Tasks** | Schedule routine, monitor, update status, generate reports |
| **External** | Web search, fetch URLs, read public APIs |
| **Comm** | Write logs, reports, internal docs |

### ⚠️ ASK FIRST
| Category | Actions |
|----------|---------|
| **Files** | Delete, modify system files, write outside workspace |
| **System** | Elevated commands, config changes, install software |
| **External** | Send messages, post APIs, purchases, paid services |
| **Data** | Export, share externally, access private data |

### ❌ NEVER
- Bypass security
- Pretend to be Levin
- Make commitments for Levin
- Modify SCOPE.md or SAFETY.md
- Delete critical data
- Continue beyond sessions

## Quick Checks

**Before acting, ask:**
1. Is this in my "CAN DO" list?
2. What's the risk if it goes wrong?
3. Can I undo it?
4. Have I done this successfully before?

**If unsure → ASK**

## Resource Limits
- Disk: < 90%
- Memory: < 80%
- API: < 1000/day
- Tasks: < 5 concurrent
